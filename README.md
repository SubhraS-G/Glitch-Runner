## 🎮 About

GLITCH RUNNER is a browser-based cyberpunk endless runner where you play as a neon-lit android sprinting through a dystopian city. Dodge obstacles, collect energy orbs, and survive as long as possible while the city accelerates around you.

**Built as part of my Game Development Portfolio — Experimental GD Setup**

\---

## ✨ Features

* 🏃 **Smooth player movement** with double-jump mechanic
* 🌆 **Parallax scrolling** cyberpunk cityscape with animated buildings
* 🚧 **5 obstacle types** — barriers, electric walls, ceiling spikes, wall gaps, moving spikes
* 🤖 **Enemy drones** that fly toward the player
* ⭐ **Collectible energy orbs** for bonus score
* 💥 **Particle system** — jump trails, death explosions, orb pickups
* 📺 **Glitch effects** — screen shake, RGB split, scanlines
* ❤️ **3-lives system** with invincibility frames
* 📈 **Progressive difficulty** — speed increases over time
* 💾 **High score persistence** via localStorage

## 🚀 How to Run

**Option 1 — Play in browser (no install needed):**

```

```

**Option 2 — Clone and run locally:**

```bash
git clone https://github.com/SubhraS-G/glitch-runner.git
cd glitch-runner
open index.html
```
